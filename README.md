# Quest Explorer

## Overview
Quest Explorer is a treasure hunt themed web interface where explorers travel through different regions, complete missions, and uncover hidden relics. The project focuses on building an engaging user interface using only HTML and CSS.

This project demonstrates semantic HTML structure, responsive layouts, CSS animations, and interactive UI elements.

Created for **CSC 4370 Web Programming – Project 1 (HTML/CSS)**.

## Pages

### Home Page
The homepage introduces the Quest Explorer world and allows users to begin exploring the adventure.

Features:
- Hero introduction section
- Feature cards describing exploration and missions
- Navigation to the world map

### World Map
The world map shows different regions explorers can visit.

Regions include:
- Jungle Temple
- Crystal Coast
- Haunted Castle
- Dark Forest

The page uses **CSS Grid** to organize the map locations visually.

### Mission Pages
Each region contains its own mission page where explorers can follow a treasure route.

Features:
- Mission route map
- Checkpoint cards
- Card flip animations
- Mission difficulty and reward information

### About Page
The about page explains the purpose of the project and introduces the developers.

About Quest Explorer

Quest Explorer is a treasure hunt themed web experience where explorers travel across mysterious regions, complete missions, and uncover hidden relics.

This project demonstrates modern web design techniques including Flexbox, CSS animations, responsive layouts, and interactive UI elements.
